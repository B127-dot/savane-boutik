# Switzer Font Installation

## Instructions
Please extract the following font files from the Switzer ZIP archive to this directory:

- Switzer-Regular.ttf
- Switzer-Medium.ttf
- Switzer-Semibold.ttf
- Switzer-Bold.ttf

These fonts are required for the BurkE-Shop design system.

## Font Weights
- Regular: 400
- Medium: 500
- Semibold: 600
- Bold: 700
